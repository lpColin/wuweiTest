## 一、产品定位与目标

**产品名称（建议）：** 五维测评系统 / 参军能力评估中心

**一句话定位：** 面向有参军意向的青少年，提供心理、体质、体能、职业适应性一体化模拟测评的训练辅助工具，帮助用户了解自身能力结构、发现短板、针对性提升。

**核心价值主张：**
- 测评全流程线上化，随时随地可练习
- 题目紧贴真实征兵/军检标准，降低考前焦虑
- 分层定价，免费试用 -> 付费解锁，降低决策门槛
- 个人数据中心化，追踪成长轨迹

**目标用户：** 18-24岁有参军/军校报考意向的青年、役前训练人群、家长/教官辅助参考

### 3.1 首页

**作用：** 流量入口，展示全部测评入口和平台价值。

**页面结构：**
- 顶部导航栏：Logo + 标题 + 用户状态（未登录显示登录注册按钮 / 已登录显示昵称头像）+ 个人中心入口
- 公告条（可关闭）：后台可配置的通知文字，滚动展示，点击关闭当天不再显示
- Banner 区：可配置的活动横幅（如 新版上线 / 限时活动），可跳转指定 URL
- 核心价值主张区：一句话介绍平台定位
- 四大能力卡片区：心理素质、文化能力、体质体能、职业适应性
- 测评模块卡片网格：共 17 个模块入口，每张卡片包含：标签（热门/必练/新增/役前）、标题、简介描述、标签集（题量/题型/时限/评级方式）、进入按钮
- 工具入口区：体检标准参考、体能评分工具、SBTI 人格测试、备考方案库
- 底部 Footer：版权声明、免责说明

**交互逻辑：**
- 未登录点击测评 -> 跳转登录页（或弹出登录 Modal）
- 未付费用户点击付费测评 -> 提示请先购买正式码
- 测评卡片点击后直接进入测评流程页

### 3.2 综合模拟测试

**说明：** 一次性完成四个子测验，输出综合评级报告。

**流程：**
  1. 欢迎页（测试说明）
  2. 第一部分：数学能力（60道计算题，15秒/题）
  3. 完成页 -> 可选继续语言测试
  4. 第二部分：语言能力（60道选择题，15秒/题）
  5. 完成页 -> 可选继续人格测验
  6. 第三部分：人格测验（150道，是/否作答，15秒/题）
  7. 完成页 -> 可选进入心理筛查，或跳过直接提交
  8. 第四部分：心理筛查（200道，是/否作答，10秒/题）【可选】
  9. 综合报告页（总评分 + 各部分得分 + 效度分析 + 提升建议）

**数学题目类型：**
- 两位数加减法（结果取最后一位填入）
- 进阶：乘法、除法、混合运算

**报告内容：**
- 总评分（A/B/C/D 四级）
- 各部分正确率百分比
- 临床量表条形图（人格部分）
- 效度量表（答题真实性检测）
- 个性化提升建议
- 警示提示（建议多次练习）
- 错题回顾（Modal 弹窗）

### 3.7 体能评分工具

**说明：** 输入体能成绩，自动根据标准换算评分，无需在线答题。

**评分项目（参考《国家学生体质健康标准》）：**

| 项目 | 输入方式 | 说明 |
|---|---|---|
| 性别切换 | 按钮切换男/女 | 评分标准不同 |
| 身高 | 数字输入（cm） | 计算 BMI |
| 体重 | 数字输入（kg） | 计算 BMI |
| BMI | 自动计算 | 超重/正常/过轻三级提示 |
| 肺活量 | 数字输入（mL） | 按性别+年龄查表评分 |
| 坐位体前屈 | 数字输入（cm） | 男子/女子不同评分表 |
| 立定跳远 | 数字输入（cm） | 按性别+年龄查表评分 |
| 跑步 | 男1000m / 女800m，时分秒输入 | 按成绩查表评分 |
| 仰卧起坐（女） | 次数 | 按年龄查表评分 |
| 引体向上（男） | 次数 | 按年龄查表评分 |
| 跳绳 | 次数/分钟 | 选做项目 |
| 平衡能力 | 闭眼单脚站立秒数 | 选做项目 |

**输出：**
- 每项成绩单列评分（优秀/良好/及格/不及格）
- 总分（加权求和）
- 总评等级（优秀/良好/及格/不及格）
- 雷达图展示各维度评分
- 评分公式展示（用户可理解计算逻辑）

### 3.11 个人中心

**说明：** 登录用户的个人信息聚合页。

**个人信息卡片：**
- 头像（首字母圆形图标）
- 昵称、手机号（脱敏）
- 学历信息
- 地区信息

### 3.12 邀请好友

**说明：** 积分裂变体系核心页面。

**功能：**
- 展示用户专属邀请码
- 专属邀请链接（URL 参数携带邀请码）
- 一键复制链接按钮
- 微信分享卡片自动预览（标题 + 描述 + 图标）
- 海报生成功能：Canvas 绘制邀请海报（含 Logo、标题、二维码），二维码内嵌邀请链接，保存图片到本地，微信长按识别

**积分规则：**
- 好友通过邀请链接注册 -> 邀请人 +20，好友 +20
- 好友完成任意一项测评 -> 邀请人 +50，好友 +50
- 同一好友仅计算一次

**邀请记录列表：** 展示被邀请人（脱敏手机号）、注册时间、是否完成测评，空状态引导分享

## 四、题库数据结构设计

### 4.1 题目表 questions

| 字段 | 类型 | 说明 |
|---|---|---|
| id | BIGINT PK | 自增主键 |
| test_module | VARCHAR(50) | 所属测验模块，如 test1, test2... |
| type | ENUM | 题型：choice/yesno/fill/multi/show |
| content | TEXT | 题目正文 |
| options | JSON | 选项列表，choice/multi时有效 [{label, content}] |
| answer | VARCHAR(255) | 正确答案，choice为字母，yesno为0/1，fill为数值 |
| explanation | TEXT | 答案解析（可选） |
| difficulty | TINYINT | 难度1-5，默认2 |
| tags | JSON | 标签，如 [数学, 代数, 两位数加法] |
| score | INT | 分值，默认1 |
| is_active | BOOLEAN | 是否启用，默认TRUE |

### 4.2 测评记录表 test_records

| 字段 | 类型 | 说明 |
|---|---|---|
| id | BIGINT PK | 自增主键 |

| test_module | VARCHAR(50) | 测验模块 |
| total_score | INT | 原始总分 |

| grade | CHAR(1) | 评级 A/B/C/D |

| scale_scores | JSON | 各维度分量表得分（如MMPI各量表） |

## 五、评分算法设计

### 5.1 A-D 四级评级（通用）

- A（优秀）：>= 85分
- B（良好）：>= 70分
- C（及格）：>= 60分
- D（不及格）：< 60分

> 各测验可自定义权重，总分 = 各部分正确题数 x 各部分满分权重 求和

### 5.2 综合模拟加权评分

- 数学正确率 x 0.30
- 语言正确率 x 0.30
- 人格测验分量表标准化得分 x 0.25
- 心理筛查得分 x 0.15

### 5.3 MMPI 人格量表评分

- 各分量表原始分 -> T分换算（常模表）
- 效度量表（L/K/F）检测答题真实性
- 临床量表：Hs/D/Hy/Pd/Mf/Pa/Pt/Sc/Ma/Si（共10个）
- 高于常模 T>70 标记异常维度
- 输出雷达图 + 各维度描述性解读

### 5.4 SCL-90 评分

- 1-5 级评分
- 总分 = 各题得分之和，阳性项目数，阳性症状痛苦水平
- 因子分 = 因子内各题总分 / 因子题数
- 各因子分与常模对比，>2.0 提示阳性

### 5.5 体能评分（查表法）

- 根据性别 + 年龄段查 physical_standards 表
- 找到对应成绩所在的分数区间（100/80/60/40/20）
- 总分 = 各项目得分 x 权重 求和，权重由标准规定
- BMI 单独计算（过轻/正常/超重/肥胖四级，不直接扣分但影响总评）

### 9.3 题目随机抽题策略（后端实现）

1. 从题库中取出该模块全部可用题目
2. 随机打乱顺序（Fisher-Yates 洗牌算法）
3. 取前 N 道（N 由模块配置决定）
4. 缓存本次抽题结果（Redis，key = sessionId，过期时间 = 测评最大时长）
5. 返回题目列表（不含答案）
