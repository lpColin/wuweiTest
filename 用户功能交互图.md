## 核心交互流程说明

### 流程1：用户登录注册

```mermaid
flowchart LR
    V(访客) --> Auth[认证注册登录] --> DB[(MySQL)]
    Auth --> Profile[个人信息设置] --> VIP[正式码激活VIP]
`

### 流程2：测评完整流程（核心）

```mermaid
flowchart LR
    U(VIP用户) --> T[选择测评模块]
    T --> E[测评执行引擎]
    E --> R[随机抽题] -.-> Redis[(Redis)]
    R --> Timer[计时器 15s/20s]
    Timer --> Submit[答题提交] --> Score[评分计算]
    Score --> DB[(MySQL)]
    Score --> Report[报告生成]
    Report --> Radar[综合雷达图]
    Report --> MMPI[MMPI量表图]
    Report --> SCL[SCL-90因子图]
    Report --> Advice[个性化建议]
`

### 流程3：邀请裂变体系

```mermaid
flowchart LR
    U1(邀请人VIP) --> Invite[邀请好友]
    Invite --> OSS[(OSS)]
    Invite --> Code[生成邀请码+二维码]
    Code --> Share[分享链接/海报]
    Share --> U2(被邀请人) --> Auth2[注册账号]
    Auth2 -.-> |注册奖励+20分| P1[积分中心]
    Auth2 --> Test[完成测评] -.-> |测评奖励+50分| P2[积分中心]
    P1 --> U1[邀请人+积分]
    P2 --> U1
`

### 流程4：VIP权限控制

```mermaid
flowchart LR
    F(免费用户) -.-> |点击付费测评| Gate{VIP权限校验}
    Gate -->|通过| V(VIP用户) --> Full(全部测评功能)
    Gate -.->|拒绝| F
    F --> Free(免费测评/积分体系)
`

### 流程5：管理员运营体系

```mermaid
flowchart LR
    Admin(管理员) --> Home[管理后台]
    Home --> UM[用户管理] -.-> |禁用启用删除| DB[(MySQL)]
    Home --> RM[成绩管理] -.-> |筛选导出| DB
    Home --> CM[正式码管理] -.-> |生成激活码| VIP[用户VIP激活]
    Home --> PM[积分码管理] -.-> |发放奖励| Points[积分中心]
    Home --> NM[公告管理] -.-> |发布公告| Notice[首页通知栏]
    Home --> AM[文章管理] -.-> |发布编辑| KB[知识库]
    Home --> Stats[数据统计] -.-> |UV/注册/测评量| DB
`

### 流程6：体能评分流程

```mermaid
flowchart LR
    U(用户) --> P[体能评分工具]
    P --> BMI[输入身高体重-BMI计算]
    BMI --> Input[输入各项体能成绩]
    Input --> Std[(体能标准表)]
    Std --> Calc[按性别年龄查表评分]
    Calc --> Score[各项目得分展示]
    Calc --> Radar[雷达图]
    Calc --> Total[总分+总评等级]
`

## 模块说明速查

| 模块分类 | 模块名称 | 访问角色 | 数据依赖 |
|---|---|---|
| 首页入口 | 首页/公告/Banner | 全部用户 | MySQL-公告表 |
| 测评中心 | 综合模拟/初检/役前/单项/SBTI | VIP | MySQL-题库,Redis-缓存 |
| 测评引擎 | 抽题/计时/评分/报告 | VIP | MySQL,Redis |
| 报告系统 | 雷达图/MMPI/SCL-90/错题 | VIP | MySQL-记录表 |
| 体能评分 | 体能工具/BMI | VIP | MySQL-标准表 |
| 用户体系 | 认证/个人信息/VIP激活 | 全部登录用户 | MySQL-用户表 |
| 积分体系 | 积分中心/邀请/商城 | 全部登录用户 | MySQL-积分表 |
| 知识库 | 备考方案/体检标准/文章 | 全部用户 | MySQL-文章表 |
| 管理后台 | 用户/成绩/码/公告/统计 | 仅管理员 | MySQL全表 |
| 数据层 | MySQL/Redis/OSS | 全部模块 | - |
