## 二、子流程图

### 流程1：用户登录注册

`mermaid
flowchart LR
    V(访客) --> Auth[认证注册登录]
    Auth -.-> |存储用户数据| DB[(MySQL)]
    Auth --> Profile[个人信息设置]
    Profile --> VIP[正式码激活VIP]
`

### 流程2：测评完整流程（核心）

`mermaid
flowchart LR
    U(VIP用户) --> T[选择测评模块]
    T --> Rand[随机抽题]
    Rand -.-> |抽题结果缓存| Redis[(Redis)]
    Rand --> Timer[计时器 15s/20s]
    Timer --> Submit[答题提交] --> Score[评分计算]
    Score -.-> |答题记录| DB[(MySQL)]
    Score --> Report[报告生成]
    Report --> Radar[综合雷达图]
    Report --> MMPI[MMPI量表图]
    Report --> SCL[SCL-90因子图]
    Report --> Advice[个性化建议]
`

### 流程3：邀请裂变体系

`mermaid
flowchart LR
    U1(邀请人VIP) --> Invite[邀请好友]
    Invite --> Code[生成邀请码+二维码]
    Code -.-> |海报存储| OSS[(OSS)]
    Code --> Share[分享链接/海报]
    Share --> U2(被邀请人)
    U2 --> Reg[注册账号]
    Reg -.-> |注册奖励+20分| P1[积分中心]
    U2 --> Test[完成测评]
    Test -.-> |测评奖励+50分| P2[积分中心]
    P1 --> Reward1[邀请人+积分]
    P2 --> Reward2[邀请人+积分]
`

### 流程4：VIP权限控制

`mermaid
flowchart LR
    F(免费用户) -.-> |点击付费测评| Gate{VIP权限校验}
    Gate -->|通过| V(VIP用户)
    V --> Full(全部测评功能 3600+题)
    Gate -.->|拒绝| F
    F --> Free(免费测评+积分体系)
`

### 流程5：管理员运营体系

`mermaid
flowchart LR
    Admin(管理员) --> Home[管理后台]
    Home --> UM[用户管理]
    UM -.-> |禁用启用删除| DB[(MySQL)]
    Home --> RM[成绩管理]
    RM -.-> |筛选导出CSV| DB
    Home --> CM[正式码管理]
    CM -.-> |生成激活码| VIP[用户VIP激活]
    Home --> PM[积分码管理]
    PM -.-> |发放奖励| Points[积分中心]
    Home --> NM[公告管理]
    NM -.-> |发布公告| Notice[首页通知栏]
    Home --> AM[文章管理]
    AM -.-> |发布编辑| KB[知识库]
    Home --> Stats[数据统计]
    Stats -.-> |UV注册测评量| DB
`

### 流程6：体能评分流程

`mermaid
flowchart LR
    U(用户) --> P[体能评分工具]
    P --> BMI[输入身高体重-BMI计算]
    BMI --> Input[输入各项体能成绩]
    Input --> Std[(体能标准表)]
    Std --> Calc[按性别年龄查表评分]
    Calc --> Score[各项目得分展示]
    Calc --> Radar[雷达图]
    Calc --> Total[总分+总评等级]
`
