## 二，子流程图

### 流程1：用户登录注册

```mermaid
flowchart LR
    V1(访客) --> A1[认证注册登录]
    A1 -.-> DB1[(MySQL)]
    A1 --> P1[个人信息设置]
    P1 --> VIP1[正式码激活VIP]
```

### 流程2：测评完整流程

```mermaid
flowchart LR
    U(VP用户) --> T[选择测评模块]
    T --> R[随机抽题]
    R -.-> Redis[(Redis)]
    R --> Timer[计时器 15s/20s]
    Timer --> Sub[答题提交] --> Score[评分计算]
    Score -.-> DB[(MySQL)]
    Score --> Rep[报告生成]
    Rep --> Radar[综合雷达图]
    Rep --> MMPI[MMPI量表图]
    Rep --> SCL[SCL90因子图]
    Rep --> Advice[个性化建议]
```

### 流程3：邀请裂变体系

```mermaid
flowchart LR
    Inv1(邀请人VP) --> Invite[邀请好友]
    Invite --> Code[生成邀请码+二维码]
    Code -.-> OSS[(OSS)]
    Code --> Share[分享链接/海报]
    Share --> Inv2(被邀请人) --> Reg[注册账号]
    Reg -.-> Pt1[积分中心 +20分]
    Inv2 --> Test[完成测评] -.-> Pt2[积分中心 +50分]
    Pt1 --> R1[邀请人+积分]
    Pt2 --> R2[邀请人+积分]
```

### 流程4：VIP权限控制

```mermaid
flowchart LR
    F(免费用户) -.-> Gate{VIP权限校验}
    Gate --> V(VP用户) --> Full(全部测评功能)
    Gate -.-> F
    F --> Free(免费测评+积分体系)
```

### 流程5：管理员运营体系

```mermaid
flowchart LR
    Admin(管理员) --> Home[管理后台]
    Home --> UM[用户管理] -.-> DB[(MySQL)]
    Home --> RM[成绩管理] -.-> DB
    Home --> CM[正式码管理] -.-> VIP[用户VP激活]
    Home --> PM[积分码管理] -.-> Points[积分中心]
    Home --> NM[公告管理] -.-> Notice[首页通知栏]
    Home --> AM[文章管理] -.-> KB[知识库]
    Home --> Stats[数据统计] -.-> DB
```

### 流程6：体能评分流程

```mermaid
flowchart LR
    U(用户) --> P[体能评分工具]
    P --> BMI[输入身高体重-BMI计算]
    BMI --> Input[输入各项体能成绩]
    Input --> Std[(体能标准表)]
    Std --> Calc[按性别年龄查表评分]
    Calc --> Score[各项目得分]
    Calc --> Radar[雷达图]
    Calc --> Total[总分+总评等级]
```

## 三、图例

| 图形 | 含义 |
|---|---|
| 箭头 --> | 正常数据流向 / 模块跳转 |
| 虚线 -.-> | 条件触发 / 权限校验 / 存储操作 |

| 圆柱 | 数据库层 |

| 黄色节点 | 管理员 |

## 四、模块速查表

| 模块分类 | 模块名称 | 访问角色 | 数据依赖 |
|---|---|---|---|
| 首页入口 | 首页 / 公告 / Banner | 全部用户 | MySQL-公告表 |
| 测评中心 | 综合模拟 / 初检 / 役前 / 单项 / SBTI | VIP | MySQL-题库, Redis-缓存 |
| 测评引擎 | 抽题 / 计时 / 评分 / 报告 | VIP | MySQL, Redis |
| 报告系统 | 雷达图 / MMPI / SCL-90 / 错题 | VIP | MySQL-记录表 |
| 体能评分 | 体能工具 / BMI | VIP | MySQL-标准表 |
| 用户体系 | 认证 / 个人信息 / VIP激活 | 全部登录用户 | MySQL-用户表 |
| 积分体系 | 积分中心 / 邀请 / 商城 | 全部登录用户 | MySQL-积分表 |
| 知识库 | 备考方案 / 体检标准 / 文章 | 全部用户 | MySQL-文章表 |
| 管理后台 | 用户 / 成绩 / 码 / 公告 / 统计 | 仅管理员 | MySQL全表 |
| 数据层 | MySQL / Redis / OSS | 全部模块 | - |
